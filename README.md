# Hy-Fi-Spleef
Hy-Fi Spleef offers a modern recreation of the classic block-breaking arena minigame for Hytale. Players break blocks beneath opponents until one survives. Supports shovel modes, layers, timed matches, and power-ups. Works perfectly as a lobby filler or queue-based minigame.
